## Getting Started

No build step. No package manager. No dependencies.

```bash
git clone https://github.com/yourusername/greenfield-academy.git
cd greenfield-academy
```

**Run locally:**

```bash
# Python
python3 -m http.server 8080

# Node
npx serve .
```

Visit `http://localhost:8080`. Opening `index.html` directly via `file://` works for most features; a local server avoids font/asset loading issues on stricter browsers.

## Design System

All visual decisions live in `variables.css` as CSS custom properties. Edit one token — it propagates everywhere.

### Color Tokens

| Token | Value | Role |
|-------|-------|------|
| `--clr-emerald` | `#1E5645` | Primary brand |
| `--clr-emerald-mid` | `#2A6E58` | Hover state |
| `--clr-terracotta` | `#C0624A` | Secondary accent |
| `--clr-ivory` | `#F9F6F0` | Page background |
| `--clr-stone-900` | `#2A2520` | Primary text |

### Typography

| Role | Font | Weights |
|------|------|---------|
| Display / Headings | Sora | 300 · 400 · 500 · 600 · 700 · 800 |
| Body / UI | Plus Jakarta Sans | 300 · 400 · 500 · 600 · 700 |

Fluid scale via `clamp()` — from `0.7rem` base to `7rem` hero. No JavaScript involved.

### Motion Tokens

| Token | Value |
|-------|-------|
| `--ease-out` | `cubic-bezier(0.22, 1, 0.36, 1)` |
| `--ease-spring` | `cubic-bezier(0.34, 1.56, 0.64, 1)` |
| `--dur-xl` | `800ms` |

## Customization

**Swap brand colors** — `variables.css`:
```css
--clr-emerald:    #1E5645;
--clr-terracotta: #C0624A;
```

**Control scroll reveal** — HTML attributes:
```html
<div data-reveal="up" data-delay="200">...</div>
<div data-reveal="left">...</div>

<div data-stagger="100">
  <div>Card 1</div>
  <div>Card 2</div>
</div>
```

**Replace images** — drop files into `assets/images/<folder>/`. Filenames are referenced directly in `index.html`.

## Responsive Breakpoints

| Breakpoint | Target |
|------------|--------|
| `max-width: 480px` | Small phones — single column, compact modals |
| `max-width: 768px` | Tablets — mobile nav active, parallax disabled |
| `max-width: 1024px` | Laptops — reduced grid columns, adjusted hero |
| `min-width: 1440px` | Large screens — expanded container, larger type |
